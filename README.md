# tth-analysis
the repository contains root macros for tth analysis. the analysised root file is Higgs CP even or odd, with Higgs Boson decays to double photons.tth_analysis is an example of plotting digrams."macro8.c" is an example of data fit and "BDT_tth_Application.C" is written for applying the results of "BDT_tth.C",which is an example of BDT method.
